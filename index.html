  • Assignment 2: JavaScript Creating a Form

    Description:

    This assignment was to create a webpage using widgets with JavaScript to collects some information from the user and based of the information to make decisions, and provide the user with feedback. For the website, we were free to do anything we wanted as long as there was the following:

    1. Textboxes to collect free form information
    2. Radio buttons to collect one option among many
    3. Check boxes to collect many options among many
    4. Drop down boxes to collect many options among many
    5. A reset button option
    6. A submit button option that will call an event

    For my webpage, I decided to go with an online coffee order form where you enter your first name, last name, contact number, and email. Then you can pick what type of coffee you would like. (i.e. Coffee, Espresso, Macchiato, Chemex, etc.) You then pick what size you would want the coffee in then, which beans to use from light, medium, dark, and decaf roast. Lastly, if you would like an extra shot, almond/soy milk, handmade syrups, and/ or honey. Also there is an text area where you can add an comment or a specific instruction for how would you like your coffee.

    This assignment has taught and refreshed me about the basics fundamentals of JavaScript. The big and main thing for me was creating the scripts because you have to type it exactly how you named it or it would not work and you have to type it a certain way.

    Objectives:

    • Become familiar with different type of JavaScript widgets. (i.e. checkboxes, radio buttons, text area, and as well as a scroll down box
    • Become familiar with creating scripts, function, and variables for JavaScript to do certain actions when hitting a submit or reset button
    • All understanding how CSS and JavaScript interact with each other when working with the textboxes, radio buttons, check boxes, drop down boxes, and the reset and submit button
